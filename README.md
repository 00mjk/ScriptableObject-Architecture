# ScriptableObject-Architecture
Makes using Scriptable Objects as a fundamental part of your architecture in Unity super easy

# Features
- Automatic Script Generation
- Variables - All C# primitives
- Variable References
- Typed Events
- Runtime Sets
- Custom Icons
